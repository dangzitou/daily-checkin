# Wife Checkin Implementation Plan

Build a family-private task check-in H5 system with Vue 3, NestJS, MySQL, Redis, SMTP reminders, and Docker Compose deployment on local port 80.

The implementation follows the approved plan from the conversation: open registration, username or numeric id login, daily user-created tasks, calendar history, streak stats, per-task email reminders, same-origin `/api` routing, and `Asia/Shanghai` date semantics.
